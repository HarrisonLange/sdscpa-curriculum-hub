# SDSCPA Curriculum Hub

A live, branded curriculum index for SDSCPA Design & Production teachers. The site reads the public curriculum workbook directly in the browser and preserves the hyperlinks stored in each workbook cell.

## Local development

```sh
pnpm install
pnpm run dev
```

## Validation

```sh
pnpm run build
```

GitHub Pages deployment runs automatically from the `main` branch.
